# HiBuddy

인하대학교 국제학생들의 정보 교류 및 친목활동과 한국어 발음 평가를 위한 커뮤니티 서비스.

## 개발 환경 & 기술 스택

- java 17.0.9 2023-10-17 LTS
- Java + Spring Boot
- AWS EC2, S3, RDS 
- Gradle
- tomcat

## 주요 기능

### 회원 기능

- 구글/카카오 소셜 로그인
- 온보딩 및 프로필 기능
- 작성/저장한 스레드 조회

- 스레드 업로드, 조회, 수정 및 삭제

- 텍스트 및 이미지 업로드
